2 시험 결과 저장 (Apps Script)

아래 코드를 구글 시트의 Apps Script에 붙여넣고 웹 앱으로 배포하면
시험 결과가 자동으로 '시험결과' 시트에 기록됩니다.
코드를 수정했다면 반드시 재배포(새 배포)해야 합니다.

function doPost(e) { try { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName('시험결과'); if (!sheet) { sheet = ss.insertSheet('시험결과'); var h = [ '타임스탬프','지점명','응시자 성명','직책','교육 단계','응시 횟수', '총점(%)','합격 여부', '브랜드(%)','서비스(%)','커피(%)','위생(%)','게임(%)','포스(%)','인사/청소(%)', '소요 시간(초)' ]; sheet.appendRow(h); sheet.getRange(1, 1, 1, h.length) .setFontWeight('bold') .setBackground('#d02b22') .setFontColor('#ffffff'); } // application/x-www-form-urlencoded 형식으로 전송된 data 파라미터를 읽음 var raw = e.parameter.data; var d = JSON.parse(raw); var c = d.cats || {}; sheet.appendRow([ d.timestamp, d.store, d.name, d.role, d.step, d.attempt, d.score, d.pass ? '합격' : '불합격', c['브랜드'] || '', c['서비스'] || '', c['커피'] || '', c['위생'] || '', c['게임'] || '', c['포스'] || '', c['인사/청소'] || '', d.duration ]); return ContentService .createTextOutput(JSON.stringify({ success: true })) .setMimeType(ContentService.MimeType.JSON); } catch (err) { return ContentService .createTextOutput(JSON.stringify({ success: false, error: err.toString() })) .setMimeType(ContentService.MimeType.JSON); } }

⚠ 코드를 바꿨다면 반드시 새 버전으로 재배포해야 합니다

최초 배포 또는 코드 수정 후 재배포:

  1. 구글 시트에서 확장 프로그램 → Apps Script 클릭
  2. 기존 코드를 전부 지우고 위 코드를 붙여넣기 → Ctrl+S 저장
  3. 배포 → 새 배포 클릭 (기존 배포 관리 X, 반드시 새 배포)
  4. 유형 선택 아이콘(⚙) → 웹 앱 선택
  5. 설명: 아무거나 입력 / 다음 사용자로 실행: 나 / 액세스 권한: 익명의 사용자 포함 모든 사용자
  6. 배포 클릭 → 권한 요청 팝업이 뜨면 액세스 허용
  7. 생성된 웹 앱 URL을 아래에 붙여넣기
💡 코드를 수정하고 저장(Ctrl+S)만 하면 기존 배포에는 반영되지 않습니다.
반드시 배포 → 새 배포를 통해 새 버전을 만들어야 합니다.
💡 연동 없이도 사용 가능합니다.
설정하지 않으면 내장 42문항으로 진행됩니다. 언제든지 이 화면에서 설정할 수 있습니다.
